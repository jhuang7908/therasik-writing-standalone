# TheraSIK 中国支付 + 发票接入申请说明书

**主体**：启元智研（北京）生物科技有限公司  
**目标**：微信支付接入 + 微信网页登录（OAuth） + 增值税电子发票自动开具

---

## 一、微信支付（已有商户，补充 API 凭证）

> 你们已有微信支付店铺，不需要重新申请商户号。
> 只需在商户平台获取开发接入所需的 API 凭证。

### 1.1 登录入口

**微信支付商户平台**：https://pay.weixin.qq.com  
登录账号：绑定商户号的微信账号

### 1.2 需要获取的信息（4项）

| 编号 | 信息名称 | 在哪里找 | 格式示例 |
|---|---|---|---|
| ① | **商户号 (mch_id)** | 首页右上角 / 账户中心 → 商户信息 | 1234567890（10位数字） |
| ② | **API v3 密钥** | 账户中心 → API 安全 → 设置 APIv3 密钥 | 32位随机字符串 |
| ③ | **API 证书文件** | 账户中心 → API 安全 → 申请 API 证书 | apiclient_cert.pem + apiclient_key.pem（两个文件） |
| ④ | **AppID** | 账户中心 → 商户信息 → "绑定的 AppID" | wx开头的字符串 |

### 1.3 操作步骤

**获取 API v3 密钥：**
1. 登录商户平台 → 账户中心 → API 安全
2. 点击「设置 APIv3 密钥」
3. 按提示设置 32 位密钥（自定义，记录保存）

**申请 API 证书：**
1. 同页面 → 「申请 API 证书」
2. 下载工具，在本地电脑生成证书
3. 会得到两个文件：
   - `apiclient_cert.pem`（证书）
   - `apiclient_key.pem`（私钥）
4. 将两个文件上传到服务器 `/etc/therasik-mcp/wechat/`

### 1.4 注意事项

- API 私钥（`apiclient_key.pem`）绝对不能提交 Git，只存服务器
- APIv3 密钥也只存 `/etc/therasik-mcp/env`，不写代码里
- 证书有效期 **5年**，到期需重新申请

## 二、微信网页登录 OAuth（网站应用，非小程序）

> 没有小程序没关系。用「网站应用」可以在 PC 浏览器实现微信扫码登录，
> 用户打开 mcp.therasik.com 注册时扫码即可，无需下载任何 App。

### 2.1 申请入口

**微信开放平台**：https://open.weixin.qq.com  
（注意：不是微信支付商户平台，是独立的开放平台）

### 2.2 申请步骤

1. **注册/登录微信开放平台**
   - 用公司微信或管理员微信登录
   - 如未注册，需提供：管理员姓名、手机号、邮箱

2. **创建「网站应用」**
   - 管理中心 → 网站应用 → 创建网站应用
   - 填写信息：
     - 应用名称：`TheraSIK`
     - 应用官网：`https://mcp.therasik.com`
     - 应用介绍：AI 学术写作辅助工具
     - 应用图标：上传 Logo（120×120 px，≤300KB）
   - 提交审核（通常 **3-7个工作日**）

3. **认证（可选但推荐）**
   - 认证后才能使用 UnionID（跨应用识别同一用户）
   - 费用：**300元/年**
   - 材料：营业执照、法人信息

4. **获取凭证**（审核通过后）

| 信息名称 | 说明 |
|---|---|
| **AppID** | wx 开头，唯一标识这个「网站应用」|
| **AppSecret** | 配合 AppID 换取用户信息，保密 |

### 2.3 配置授权回调域

审核通过后，在应用详情设置：
- 授权回调域：`mcp.therasik.com`（不加 https://，不加路径）

### 2.4 登录效果

用户在 Chrome / Edge 打开注册页 → 点击「微信登录」  
→ 弹出二维码 → 手机微信扫码 → 授权 → 自动登录 Dashboard

## 三、增值税电子发票（诺诺网）

> 付款后自动给用户开发票，发送至邮箱。

### 3.1 申请入口

**诺诺网企业版**：https://www.nuonuo.com  
（阿里巴巴旗下，接入了微信支付和支付宝）

### 3.2 开票主体信息（需要准备）

| 信息 | 内容 |
|---|---|
| 公司名称 | 启元智研（北京）生物科技有限公司 |
| **纳税人识别号（税号）** | 营业执照上的 18 位统一社会信用代码 |
| 开票类目 | **软件服务**（税率 6%） |
| 开票银行 | 公司对公银行账号及开户行名称 |
| 地址电话 | 公司注册地址 + 联系电话 |

### 3.3 注册步骤

1. 打开 https://www.nuonuo.com → 企业注册
2. 绑定税盘或申请「无盘开票」（小微企业推荐无盘）
3. 联系诺诺客服申请 **开放平台 API 接入**
4. 获取：
   - `nuonuo_app_key`
   - `nuonuo_app_secret`
   - `nuonuo_merchant_no`（开票方商户号）

### 3.4 开票流程（接入后自动）

```
用户付款成功
    ↓
服务器调用诺诺 API /nuonuo/open/api/issuedInvoice
    ↓
传入：购买方名称/税号（用户填写） + 商品名"软件服务费" + 金额
    ↓
诺诺自动开具电子普通发票（或专用发票）
    ↓
PDF 发送至用户邮箱（5分钟内）
```

## 四、信息汇总清单（填写后发给开发）

接入完成后，将以下信息填入服务器 `/etc/therasik-mcp/env`：

```bash
# ── 微信支付 ──────────────────────────────────
WECHAT_PAY_MCH_ID=          # 商户号（10位）
WECHAT_PAY_API_V3_KEY=      # APIv3 密钥（32位）
WECHAT_PAY_APP_ID=          # 绑定的 AppID
# 证书文件路径（已上传服务器）
WECHAT_PAY_CERT_PATH=/etc/therasik-mcp/wechat/apiclient_cert.pem
WECHAT_PAY_KEY_PATH=/etc/therasik-mcp/wechat/apiclient_key.pem

# ── 微信 OAuth 登录 ───────────────────────────
WECHAT_OAUTH_APP_ID=        # 网站应用 AppID
WECHAT_OAUTH_APP_SECRET=    # 网站应用 AppSecret

# ── 诺诺发票 ──────────────────────────────────
NUONUO_APP_KEY=
NUONUO_APP_SECRET=
NUONUO_MERCHANT_NO=
NUONUO_TAX_NO=              # 公司税号（18位统一社会信用代码）
```

## 六、支付宝（可选，同时做）

如需同时支持支付宝：

1. 登录蚂蚁商家平台：https://b.alipay.com
2. 应用 → 创建应用 → 「网页&移动应用」
3. 获取 `ALIPAY_APP_ID` + `ALIPAY_PRIVATE_KEY`
4. 上传公钥，获取支付宝公钥

代码框架也已预留支付宝接入位置，时间约 **1天**。
