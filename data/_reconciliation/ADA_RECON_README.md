## Naming Drift Note

Files named "136" consistently contain more than 136 rows. The number "136" refers to the initial cohort definition. Later curation batches added entries without renaming the files. Use actual row counts from CSV reads as authoritative population sizes.

## Design Rule: Positive vs Negative Separation

`in_both_positive_and_negative = 0`: No antibody appears in both the immunogenicity panel (positive cohort) and the negative library simultaneously. This is the expected and correct state; no action needed.
