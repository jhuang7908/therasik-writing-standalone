# Core principles (from nature-paper2ppt — adopted)

## Spine (7 questions — do not copy section order)

1. Why does this problem matter?
2. What gap does the paper address?
3. What did the authors do?
4. What is the key evidence?
5. Why trust the result?
6. What is new / reusable?
7. Boundaries and open questions?

## Lean mode (default)

- Read only material needed for the argument
- Extract only figures/tables that appear in deck
- **Primary deliverable:** editable `.pptx` + `.qa.json`
- No full OCR unless PDF is scanned
- No long markdown script when user needs deck only

## Language

Default **EN** for InSynBio/Therasik scientific decks unless `--lang zh`.  
Preserve gene/platform/stat terms; use Terminology Ledger.

## QA

At least one self-audit pass; structural PPTX checks via `insynbio_paper2ppt.py`.
