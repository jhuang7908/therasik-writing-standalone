## Paper stats outputs

This folder is generated by scripts and stores **auditable statistical outputs** (TSV) used in the manuscript.

### Generate (19 clinical VHH paper figures + stats)

From repo root:

```bash
python scripts/vhh19_generate_paper_figures.py
```

### Inputs

- `paper/tables/Table1_slice3_19_clinical_vhh_master.csv`

### Outputs (examples)

- `VHH19_fold_strategy_fisher.tsv`: Fisher exact test + odds ratio (+ CI) for fold–strategy association
- `VHH19_strategy_group_kruskal.tsv`: Kruskal-Wallis tests for key metrics across strategies
- `VHH19_pairwise_mannwhitney_*.tsv`: Pairwise strategy comparisons with Holm correction
- `VHH19_spearman_human_identity_vs_B_total_1pct.tsv`: correlation result used in Fig3C
