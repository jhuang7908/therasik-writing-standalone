## 📋 TODO Before Submission

### A. Manuscript Formatting
- [ ] Convert `.md` to **DOCX/PDF** using Pandoc or Word
- [ ] Ensure figure citations are correctly numbered (Figure 1A, 1B, 2A-D)
- [ ] Double-check all P-values and statistical results match analysis reports
- [ ] Verify all in-text citations match the References section

### B. Figure Preparation
- [ ] Ensure all figures are **high-resolution** (minimum 300 DPI for print)
- [ ] Label all figure panels (A, B, C, D) clearly
- [ ] Add scale bars or axis labels where needed
- [ ] Prepare **Figure Legends** as a separate document (or within manuscript)

### C. Tables
- [ ] Convert CSV tables to **formatted Word/Excel tables**
- [ ] Add table titles and footnotes
- [ ] Ensure all abbreviations are explained in footnotes

### D. Supplementary Materials
- [ ] Create a **Supplementary Information PDF** containing:
  - Extended Methods (if needed)
  - Supplementary Tables (TableS1)
  - Supplementary Figures (if any)
  - Data file descriptions

### E. References
- [ ] Verify all 12 references are correctly cited (author names, year, journal, volume, pages)
- [ ] Replace placeholder "[Citation Needed]" with actual references
- [ ] Format references according to *Antibodies* style (Vancouver or Author-Date)
- [ ] Consider adding 3-5 more recent references (2021-2026) on VHH engineering or clinical trials

### F. Author Contributions & Acknowledgments
- [ ] Add **Author Contributions** section (CRediT taxonomy recommended)
- [ ] Add **Acknowledgments** (funding sources, technical support)
- [ ] Add **Conflicts of Interest** statement (if none, state "The authors declare no conflict of interest")
- [ ] Add **Funding** statement (grant numbers, if applicable)

### G. Cover Letter
- [ ] Draft a cover letter to the Editor highlighting:
  - Novelty: First systematic analysis of clinical VHH humanization patterns
  - Impact: Provides rational design rules for antibody engineers
  - Fit: Aligns with journal's focus on antibody therapeutics and engineering

### H. Ethics & Data Statements
- [ ] Confirm all sequences are publicly available (Thera-SAbDab is open-access)
- [ ] No ethics approval required (computational study, no patient data)
- [ ] Data availability: All data deposited or provided as supplementary files
