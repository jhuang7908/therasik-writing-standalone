📖 Format citations in Word with Zotero (like EndNote)
The ZIP contains three files:
📄 title_page.docx — Title page (authors, affiliations, declarations)
📄 manuscript.docx — Blind manuscript (line numbers, running head, references, figure legends)
📋 references.ris — Import into Zotero / EndNote / Mendeley
Manage citations in Word (4 steps):
  1. Download Zotero (free) — installs the Word plugin on Windows/Mac
  2. In Zotero: File → Import → references.ris
  3. Open manuscript.docx in Word; use Zotero tab → Add/Edit Citation
  4. Change journal style: Document Preferences → pick a CSL style (Nature, Cell, NEJM, APA…) to reformat in-text citations and the reference list
📌 Figures: prepare images in PowerPoint or Illustrator and upload each figure separately in the journal submission system (TIFF/EPS ≥300 dpi). Figure legends are appended at the end of manuscript.docx.
