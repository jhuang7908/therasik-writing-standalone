Grant workflow (recommended)

  1. Technical route map → draw.io + Bioicons extension
  2. Mechanism / antibody cartoon → Inkscape + server SVG assets
  3. Export PDF or 300 dpi PNG; paste attribution (see card below)
  4. Insert figures in M5 Grant; legends in M2 Writing

Publication checklist

  • Vector PDF/SVG preferred; raster ≥300 dpi
  • One font family; 8–10 pt labels at print size
  • CC BY (Servier): credit in legend or acknowledgments
  • Mechanism diagrams must match protocol text (Lab SOP / grant aims)
