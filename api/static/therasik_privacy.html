隐私政策

最后更新：2026 年 5 月

一、收集的信息

Therasik 控制台为提供服务，会收集以下信息：

  • 账户信息：用户名、邮箱地址、邮箱验证状态、注册时间、注册 IP 区域。
  • 账单与积分：Credit 余额、消费明细、充值/优惠码使用记录。
  • 使用数据：登录日志、任务调用记录（不含序列内容本体）、错误日志。
  • 用户提交数据：用户主动上传的抗体序列、PDB、Excel、报告等分析输入。

二、信息使用方式

账户与使用信息用于：身份认证、邮箱验证、服务访问授权、积分核算、运营支持与安全审计。AI 助手交互内容用于即时响应用户请求，不用于训练 Therasik 任何模型。

三、服务邮件

以下邮件为运营服务必需，不可单独退订：邮箱验证码、密码重置、账户安全通知、账单与积分通知。

四、可选产品更新与关联企业新闻

注册时可选勾选「接收产品更新通知」和「接受关联企业新闻服务」。勾选后，您同意 Therasik 及其关联企业（包括 InSynBio 英矽生信生态及合作伙伴）通过邮件发送：版本更新、新功能、技术白皮书、行业案例、活动邀请等内容。

您可随时通过邮件底部的「退订」链接或联系 contact@therasik.com 取消订阅。退订仅影响营销类邮件，不影响必要的服务邮件。

五、用户上传数据的处理

  • 序列与项目数据在计算过程中暂存于受访问控制的服务器，仅用于完成用户请求的分析任务。
  • 计算结果（JSON / HTML 报告）保存在用户账户下，用户可自主下载或删除。
  • 未经用户授权，Therasik 不会将用户提交的序列、结构、项目数据用于训练模型或对外披露。
  • 对于涉及商业机密的项目，建议用户与 Therasik 团队签署单独的保密协议。

六、数据安全

  • 密码采用加盐哈希存储，平台运营人员无法查看明文密码。
  • 登录会话通过签名令牌管理，支持多设备会话审计。
  • HTTPS 全链路加密传输，关键操作要求二次验证。

七、用户权利

您可以随时通过控制台或联系客服：查询账户信息、修改个人资料、导出历史报告、注销账户。注销后，除法律法规要求保留的财务记录外，账户数据将在 30 天内清除。

八、第三方服务

Therasik 控制台可能集成第三方服务（如邮件投递、支付处理、对象存储），这些服务仅获取完成其功能所必需的最小信息。

九、未成年人

本服务面向科研与企业用户。如发现未成年人未经监护人同意创建账户，请联系客服删除相关数据。

十、联系方式

隐私问题或数据请求，请发送邮件至 contact@therasik.com。
